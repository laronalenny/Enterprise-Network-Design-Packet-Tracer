#Enterprise-Network-Design (Packet Tracer Project)

## Overview
This project demonstrates the design and implementation of a structured enterprise network using Cisco Packet Tracer. It focuses on segmentation, scalability, and structured IP addressing for a small-to-medium business environment.

The network is designed to support multiple departments with secure and efficient communication using VLAN segmentation and logical IP planning.


## Objectives
- Design a scalable enterprise network topology
- Implement VLAN segmentation for department isolation
- Develop a structured IP addressing scheme
- Define device roles and responsibilities
- Document the full network design process


## Tools Used
- Cisco Packet Tracer
- Cisco routers and switches (simulated)
- Networking concepts: VLANs, subnetting, routing
- Microsoft Excel (budget planning)
